# 性能专项检查清单

## 数据获取与请求
- [ ] 是否存在 N+1 查询/请求模式
- [ ] 是否有可合并或可缓存的重复请求
- [ ] 是否正确使用服务端聚合，减少客户端串行请求

## 组件与渲染
- [ ] 是否存在不必要的客户端组件（可改为 Server Component）
- [ ] 列表渲染是否有稳定 `key` 且避免重复计算
- [ ] 是否存在明显的重复渲染来源（不必要的状态拆分、effect 链式更新）

## 资源与体积
- [ ] 是否引入了过重依赖且未按需使用
- [ ] 图片/静态资源是否优化（尺寸、格式、加载策略）
- [ ] 是否有可延迟加载或分包加载的页面模块
