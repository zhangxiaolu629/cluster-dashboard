# 安全专项检查清单

## 输入与输出安全

- [ ] 用户输入是否做了校验（类型、长度、枚举值、必填）
- [ ] 输出到页面前是否存在 XSS 风险（`dangerouslySetInnerHTML`、未转义 HTML）
- [ ] API 参数是否在服务端再次校验（不要只依赖前端校验）

## 敏感信息与凭证

- [ ] API 密钥、Token、账号密码是否通过环境变量注入
- [ ] 仓库内是否出现敏感信息硬编码（密钥、域名凭证、私有路径）
- [ ] 错误日志中是否泄露敏感字段

## 权限与边界

- [ ] 服务端接口是否做了必要的鉴权/授权校验
- [ ] 是否将仅服务端可见的数据暴露给客户端
- [ ] 是否有不必要的宽权限调用（过宽的查询或写操作）
